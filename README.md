# 🧠 BrainBlast – Adaptive Quiz Web Application

An interactive web-based quiz platform designed for Senior High School (SHS) students in the Philippines. BrainBlast enhances learning through gamification, adaptive difficulty, and performance insights.

---

## 🚀 Live Demo

🔗 https://brainblast-quiz.vercel.app/

---

## 📌 Overview

BrainBlast is a browser-based quiz application that delivers a dynamic and engaging learning experience. It adapts question difficulty based on user performance and integrates game mechanics such as streak scoring, achievements, and real-time feedback.

The platform is designed to promote active learning, critical thinking, and self-assessment in a fun and competitive environment.

---

## ✨ Features

- Multiple Game Modes (SHS, General Trivia, Mixed Mode)
- Adaptive Difficulty System
- Streak-Based Scoring & Time Bonuses
- Achievements & Leaderboard
- Performance Analysis & Feedback
- Academic Strand Recommendations (STEM, HUMSS, ABM, TVL)
- Local Storage Data Saving
- Fully Browser-Based (No Backend Required)

## ⚙️ How to Run Locally

1. Go to this repo:  
   https://github.com/devbynielll/brainblast-adaptive-quiz-website  

2. Click **Code → Download ZIP**

3. Extract the ZIP file  

4. Open in **Visual Studio Code**

5. Open `index.html` in browser  
   OR use **Live Server**
